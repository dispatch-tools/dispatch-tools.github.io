Release notes

Download Dispatcher 2.0

Version 2.0 adds faster destination awareness, repeat-download correction, better Recent Dispatches context, and clearer Help.

Download Dispatcher 2.0 highlights

  • Workspace Badge Codes: show the active workspace on the browser toolbar.
  • Toolbar Destination Badge: verify destination state without opening the popup.
  • Dispatch Again: repeat the last eligible web download and dispatch it to the current destination.
  • Recent Dispatches labeling: Dispatch Again activity is labeled clearly.
  • Help readability: clearer day-to-day guidance and common tasks.
  • Package cleanup: release candidate cleanup and store-readiness checks.
