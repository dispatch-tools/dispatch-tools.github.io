How to play

Rotate blocks until the live path connects from S to E. Start is rotatable. The End block stays locked until the live path is adjacent, then its arrow must face the same flow direction as the path entering it.

Scoring

Start with 1000 points. Hints cost 75, misclicks cost 25, and repeat rotations cost 10. Speed can add or subtract points. Long routes qualify at 200%+ of shortest path.
