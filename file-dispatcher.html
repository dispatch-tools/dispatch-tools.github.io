Future desktop utility

File Dispatcher

File Dispatcher is a related desktop utility concept for organizing local files and watched folders. It is separate from Download Dispatcher and is not required to use the browser extension.

Status

File Dispatcher is planned as a companion product area for local file organization workflows. More information will be added here when it is ready for public release.

For now, Download Dispatcher remains the current published browser extension product.
